
```bash
python3 scripts/assemble_gifs.py
```

This will:
- Process all frames (crop, resize, add transparency)
- Resize to 111x101 pixels
- Assemble into animated GIFs at 8 FPS
- Save to `gifs/` directory
- Takes less than 1 minute
